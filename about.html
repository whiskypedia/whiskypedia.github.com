About Whiskypedia

This site is run by whisky enthusiasts, Steve and Aaron. We hope to demystify, popularize, and, if nothing else, explain all about the drinking of (single malt) whiskies.

A bottle of whisky can be a big investment, and the prices have only gone in one direction. If we can help a few people find the right bottle and miss a few they wouldn't have liked, then we've succeeded. And if we can bring some new whisky drinkers up to speed on the habits and rituals around whisky, then it's all the better.

FAQ

If you're going to ask one of these things, maybe we can save you time.

  • Can you suggest a single malt? We can't answer every one of these. We're working on answering these by creating content on the website. Hopefully we'll be able to answer that question soon enough.
  • Should you drink single malts with ice? No. It's a horrible thing to do. Unless you like it, then feel free. We don't really care. Just keep the ice away from our glasses.
  • What's the difference between all the different kinds of whisky? We'll write a post and link it from here.
